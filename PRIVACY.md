# DiffShield Privacy Policy

**Last updated:** February 2026

## Data Collection

DiffShield only accesses markdown files in pull requests to provide documentation reviews. We do not store, share, or sell any code or user data.

## Data Usage

- **Code Access:** DiffShield reads only markdown (.md) files in pull requests to analyze documentation changes.
- **No Storage:** Code and documentation content is processed in real-time and is not stored on our servers.
- **No Third-Party Sharing:** User data is never shared with or sold to third parties.

## Third-Party Services

- **Hosting:** DiffShield is hosted on Render.com
- **AI Processing:** AI summaries are generated using MiniMax API

## User Rights

Users can uninstall DiffShield at any time from their GitHub repository settings. Uninstalling immediately revokes all access.
